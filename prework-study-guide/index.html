Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1-h6 decreases in size of headers by tier.
  • li lists bullet points, while ul represents unordered list and ol as ordered list.
  • section represents semantic elements that define a section in a document that contains thematic content.
  • Class works similarily to the id.
  • Physical parts of the website that can be altered have a starting brackets, and ending brackets with a slash.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width is left and right dimensions
  • Height is up and down.
  • Background color is the backdrop.
  • Color represents color of font.
  • Text align represents where font will be placed.
  • Box shadow represents shading underneath a boxed area or card.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pwd: present working directory
  • ls: list contents of current working directory
  • cd: change directory
  • git clone (ssh key): clones repo on local device
  • git init: initializes your local repo as a new git repo
  • git status: shows current status of repo
  • git add .: takes all unstaged work and stages it ready to be committed
  • git commit -m "write commit message here": commits all staged work
